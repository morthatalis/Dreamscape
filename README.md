# About
Dreamscape, a roblox website remake.  
All usable as html files.  
This remake doesnt use any credentials to operate (aka you *cant* log into this website).  
This doesn't spy on you, like how the roblox website spies on you.  
# Features (currently)
Recently played (saved on your browser),  
Games page & Game Page(both very basic),  
UserID saving, (to say "Welcome, User!" like the good old days. and in the future Friends list).
